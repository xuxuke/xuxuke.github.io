网页添加书签怎么添加图片

添加图片到书签栏

在 Chrome 和 Firefox 等现代浏览器中，将网页添加到书签栏时，您可以选择添加一张图片作为视觉参考。这可以帮助您快速识别书签并一目了然地识别内容。

Chrome

1. 打开您要添加为书签的网页。

3. 在出现的对话框中，选中“添加图片”复选框。

4. 选择要作为书签图片的网页截图或上传自己的图像。

5. 单击“保存”。

Firefox

1. 打开您要添加为书签的网页。

3. 在出现的菜单中，选择“添加书签”。

4. 在“编辑书签”对话框中，选中“添加预览图”复选框。

5. 选择要作为书签图片的网页截图或上传自己的图像。

6. 单击“保存”。

从网页拖放图片

另一种将图片添加到书签栏的方法是从网页本身拖放。

所有支持拖放的浏览器

1. 打开您要添加为书签的网页。

2. 找到要作为书签图片的图像。

3. 单击并按住图像，将其拖动到浏览器的书签栏上。

4. 松开鼠标按钮。

更改书签图片

添加书签后，您可以随时更改其图片。

Chrome

1. 右键单击书签栏中的书签。

2. 选择“编辑”。

3. 选中“添加图片”复选框。

4. 选择要作为书签图片的新图像。

5. 单击“保存”。

Firefox

1. 右键单击书签栏中的书签。

2. 选择“编辑书签”。

3. 选中“添加预览图”复选框。

4. 选择要作为书签图片的新图像。

5. 单击“保存”。

浏览器书签怎么添加

以下步骤适用于最常用的浏览器：

Chrome

1. 打开 Chrome 浏览器。

2. 导航到要添加为书签的网站。

4. 选择一个文件夹来保存书签（可选）。

5. 单击“完成”按钮。

Firefox

1. 打开 Firefox 浏览器。

2. 导航到要添加为书签的网站。

4. 选择一个文件夹来保存书签（可选）。

5. 单击“添加书签”按钮。

Microsoft Edge

1. 打开 Microsoft Edge 浏览器。

2. 导航到要添加为书签的网站。

4. 选择一个文件夹来保存书签（可选）。

5. 单击“添加”按钮。

Safari

1. 打开 Safari 浏览器。

2. 导航到要添加为书签的网站。

3. 单击书签栏中的“+”按钮。

4. 输入书签的名称和（可选）文件夹。

5. 单击“添加”按钮。

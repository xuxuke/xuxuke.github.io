从 HTML 导入书签

步骤

1. 导出 HTML 书签文件

在浏览器中，打开书签管理器。

单击“文件”菜单并选择“导出”>“HTML 文件”。

保存文件到计算机上的某个位置。

2. 导入 HTML 书签文件

Firefox

打开 Firefox 浏览器。

单击“书签”菜单并选择“整理”。

在出现的窗口中，单击“导入和备份”按钮。

选择“从 HTML 文件导入”。

导航到导出的 HTML 文件并将其打开。

Chrome

打开 Chrome 浏览器。

单击“组织器”按钮（右上角带有三条横线的图标）。

选择“导入书签”。

导航到导出的 HTML 文件并将其打开。

3. 合并导入的书签

导入的书签将与您现有的书签合并。如果您在导入前已拥有重复的书签，则导入的书签将覆盖现有书签。

注意

如果您遇到问题，请尝试关闭浏览器并重新启动。

导入书签可能会覆盖您现有的书签设置。建议在导入前备份书签。

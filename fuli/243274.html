如何进入隐藏相册

iOS 设备

1. 打开“照片”应用。

2. 点击屏幕右下角的“相册”标签。

3. 向下滚动并选择“隐藏”。

4. 输入您的设备密码或使用面容 ID 或触控 ID 解锁隐藏相册。

Android 设备

方法 1：

1. 打开“照片”应用。

2. 点击屏幕左上角的三个横线图标。

3. 选择“设置”。

4. 启用“显示隐藏相册”。

方法 2：

1. 打开文件管理器应用。

2. 导航到以下路径：/storage/emulated/0/Android/data/com.google.android.apps.photos/files/picasa_hidden

3. 此文件夹包含您隐藏的所有照片。

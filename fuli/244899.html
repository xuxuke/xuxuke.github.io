如何将网页添加到书签

Chrome

1. 打开您要保存的网页。

3. 输入文件夹名称（可选）。

4. 单击“完成”。

Firefox

1. 打开您要保存的网页。

3. 选择“书签文件夹”（可选）。

4. 单击“保存”。

Safari

1. 打开您要保存的网页。

3. 选择“添加到书签”。

4. 输入文件夹名称（可选）。

5. 单击“添加”。

Microsoft Edge

1. 打开您要保存的网页。

3. 选择“添加当前页面”。

4. 输入文件夹名称（可选）。

5. 单击“添加”。

其他浏览器

不同的浏览器可能会有略微不同的步骤，但一般流程如下：

1. 打开您要保存的网页。

2. 找到书签按钮（通常是一个星形图标）。

3. 单击书签按钮。

4. 输入文件夹名称（可选）。

5. 单击“保存”或“添加”。

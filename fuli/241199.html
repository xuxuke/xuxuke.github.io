点击进入系统安装部署页 - 哆咪影视导航

在您订购的 AWS 区域中，打开 Amazon EC2 控制台。

在导航窗格中，选择 实例。

单击 启动实例 按钮。

步骤 2：选择 Amazon Machine Image (AMI)

在 选择 Amazon Machine Image (AMI) 部分中，展开 我的 AMI。

单击 选择 按钮。

步骤 3：选择实例类型

在 选择实例类型 部分中，选择满足您性能需求的实例类型。

单击 配置详细信息 按钮。

步骤 4：配置实例详细信息

网络配置

为实例选择一个子网。

选择一个安全组或创建一个新安全组。

存储

选择根卷的类型和大小。

可选：添加其他数据卷。

标签

可选：添加标签以组织和标识您的实例。

步骤 5：购买选项

在 购买选项 部分中，选择购买选项。

单击 启动实例 按钮。

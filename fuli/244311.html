网页书签怎么设置

PC端

Google Chrome:

1. 打开Google Chrome浏览器。

2. 找到您要创建书签的网页。

4. 在弹出的窗口中，选择一个书签文件夹或创建一个新文件夹。

5. 输入书签的名称，然后单击“完成”。

Mozilla Firefox:

1. 打开Mozilla Firefox浏览器。

2. 找到您要创建书签的网页。

4. 在弹出的窗口中，选择一个书签文件夹或创建一个新文件夹。

5. 输入书签的名称，然后单击“添加书签”。

Microsoft Edge:

1. 打开Microsoft Edge浏览器。

2. 找到您要创建书签的网页。

4. 在弹出的菜单中，选择“添加到收藏夹”。

5. 选择一个书签文件夹或创建一个新文件夹。

6. 输入书签的名称，然后单击“添加”。

移动端

iOS:

1. 打开Safari浏览器。

2. 找到您要创建书签的网页。

4. 点击“添加书签”按钮。

5. 为书签输入一个名称，然后选择一个文件夹。

6. 点击“保存”。

Android:

1. 打开Google Chrome浏览器。

2. 找到您要创建书签的网页。

3. 点击菜单图标（三个点）。

4. 选择“添加到书签”。

5. 为书签输入一个名称，然后选择一个文件夹。

6. 点击“保存”。

进入隐藏入口的快捷键

Windows

显示隐藏文件和文件夹

文件资源管理器：按 Alt + T，然后选择 文件夹选项。

在 查看 选项卡上，在 高级设置 下，选中 显示隐藏的文件、文件夹和驱动器。

进入隐藏文件夹

按 Windows 键 + R，输入 %appdata%\Local，然后按 Enter。

在 Local 文件夹中，双击 Google\Chrome\User Data 文件夹。

这是 Chrome 隐藏用户数据的位置，其中包含配置文件和缓存等文件。

删除隐藏文件和文件夹

选中文件或文件夹，按 Shift + Delete。

macOS

显示隐藏文件和文件夹

打开 Finder。

在菜单栏中，按住 Option 键，然后单击 前往。

选择 库。

进入隐藏文件夹

在 Finder 中，按 Command + Shift + G。

输入路径：~/Library/Application Support/Google/Chrome

这是 Chrome 隐藏用户数据的位置，其中包含配置文件和缓存等文件。

删除隐藏文件和文件夹

选中文件或文件夹，按 Command + Delete。

跳转书签功能

概念

跳转书签功能是一种在 Markdown 文档中创建可跳转的书签并快速导航到它们的机制。它允许用户轻松地跳转到文档中的特定部分，而无需手动滚动或搜索。

用法

在 Markdown 文档中使用跳转书签功能，可以使用以下语法：

[书签名称](书签标签)

其中：

`书签名称`是书签的显示名称，当鼠标悬停在书签上时会显示。

`书签标签`是唯一标识符，用于链接到书签。

例如：

标题

[目录](目录-标签)

内容

[结论](结论-标签)

导航到书签

要导航到书签，请单击或点击书签链接。这将使文档滚动到书签位置，并自动选择书签。

键盘快捷键

许多文本编辑器和 Markdown 渲染器都提供键盘快捷键来快速导航到书签。常见的快捷键包括：

Cmd / Ctrl + P

Cmd / Ctrl + [书签标签]

好处

跳转书签功能提供了以下好处：

快速导航：允许用户直接跳转到文档中的特定部分，从而提高效率。

轻松定位：无需手动滚动或搜索，书签可以轻松定位文档中的关键点。

文档可访问性：对于较长的文档，跳转书签可以提高可访问性，并允许用户快速找到所需的信息。

注意事项

书签标签必须唯一：在同一文档中不能使用相同的书签标签。

书签链接必须与书签标签相对应：点击或单击书签链接应跳转到正确的书签位置。

书签名称应清晰简洁：书签名称应清楚地描述书签所在文档中的内容。

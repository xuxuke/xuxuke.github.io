漂亮书签怎么折叠

折叠书签是一种既实用又美观的装饰品。通过简单的折叠技巧，你可以制作出各种形状和设计的书签，让你的阅读时光更加有乐趣。

制作直角书签

所需材料：

一张方形纸

步骤：

1. 将纸张对角线对折，形成一个三角形。

2. 再将三角形对半对折，形成一个更小的三角形。

3. 将底边的两个角向上对折，使其与对折中心线对齐。

4. 将三角形的尖端向下对折，与底边对齐。

5. 打开书签，即可得到一个直角书签。

制作三角书签

所需材料：

一张矩形纸

步骤：

1. 将纸张沿长边对折。

2. 再将纸张沿短边对折，形成一个长方形。

3. 将长方形底边的两个角向上对折，使其与对折中心线对齐。

4. 将多余的部分折到三角形的后面。

5. 打开书签，即可得到一个三角书签。

制作圆形书签

所需材料：

一张圆形纸

步骤：

1. 将纸张对半对折，形成一个半圆形。

2. 再将半圆形对半对折，形成一个四分之一圆形。

3. 将四分之一圆形的一边向上对折，与对折中心线对齐。

4. 将另一边也向上对折，与中心线对齐。

5. 将半圆形打开，即可得到一个圆形书签。

制作花朵书签

所需材料：

一张方形纸

步骤：

1. 将纸张对角线对折，形成一个三角形。

2. 将三角形的底边向上对折，使其与对折中心线对齐。

3. 将多余的部分折到三角形的后面。

4. 再将三角形的两个角向上对折，形成花瓣。

5. 打开书签，即可得到一个花朵书签。

这些只是制作漂亮的书签的几种方法。你还可以发挥创造力，设计出更多独特和个性化的书签。

Word 如何快捷跳转书签

使用“导航窗格”跳转书签

1. 在 Word 文档中，点击“视图”选项卡。

2. 在“显示”组中，勾选“导航窗格”。

3. 在导航窗格中，展开“书签”部分。

4. 点击要跳转到的书签名称。文档将自动跳转到该书签所在位置。

使用“查找和替换”跳转书签

1. 点击“开始”选项卡。

2. 在“编辑”组中，点击“查找”。

3. 在“查找和替换”对话框中，切换到“转到”选项卡。

4. 在“项目”下拉列表中，选择“书签”。

5. 输入要跳转到的书签名称。

6. 点击“转到”按钮。文档将自动跳转到该书签所在位置。

使用键盘快捷键跳转书签

使用以下键盘快捷键可快速跳转书签：

Ctrl + G：打开“转到”对话框。

Ctrl + Shift + G：打开“高级查找”对话框。

F5：打开“查找和替换”对话框。

Alt + F11：打开宏窗口，并在“模块”列表中输入以下 VBA 代码：

Sub GotoBookmark(bookmarkName As String)

Dim bmk As Bookmark

Set bmk = ActiveDocument.Bookmarks(bookmarkName)

bmk.Range.Select

End If

End Sub

将 `bookmarkName` 替换为要跳转到的书签名称。

按 F5 运行宏。文档将自动跳转到该书签所在位置。

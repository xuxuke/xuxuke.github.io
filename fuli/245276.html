浏览器书签设置指南

Chrome 书签设置

步骤 1：打开书签管理器

按下 Ctrl + Shift + O (Windows) 或 Cmd + Shift + O (Mac)

或者，单击 Chrome 菜单栏中的“书签”，然后选择“书签管理器”。

步骤 2：添加书签

导航到要设置书签的网页。

为书签输入一个名称并选择一个文件夹。

单击“保存”。

步骤 3：管理书签

在书签管理器中，您可以查看、编辑、删除和整理您的书签。

要编辑书签，请右键单击它并选择“编辑”。

要删除书签，请右键单击它并选择“删除”。

Firefox 书签设置

步骤 1：打开书签管理器

按下 Ctrl + Shift + B (Windows) 或 Cmd + Shift + B (Mac)

或者，单击 Firefox 菜单栏中的“书签”，然后选择“显示所有书签”。

步骤 2：添加书签

导航到要设置书签的网页。

为书签输入一个名称并选择一个文件夹。

单击“保存”。

步骤 3：管理书签

在书签管理器中，您可以查看、编辑、删除和整理您的书签。

要编辑书签，请右键单击它并选择“属性”。

要删除书签，请右键单击它并选择“删除”。

Microsoft Edge 书签设置

步骤 1：打开收藏夹面板

或者，按 Ctrl + Shift + C (Windows) 或 Cmd + Shift + C (Mac)

步骤 2：添加书签

导航到要设置书签的网页。

单击收藏夹面板中的“添加到收藏夹”按钮。

为书签输入一个名称并选择一个文件夹。

单击“保存”。

步骤 3：管理书签

在收藏夹面板中，您可以查看、编辑、删除和整理您的书签。

要编辑书签，请右键单击它并选择“编辑收藏夹”。

要删除书签，请右键单击它并选择“删除”。

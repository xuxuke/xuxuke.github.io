DNS 解析

建立网络连接

一旦 DNS 解析完成，浏览器就会与目标服务器建立一个 TCP（传输控制协议）连接。TCP 是一种可靠且有序的传输协议，它确保数据能够完整且无差错地传输。

服务器响应

解析 HTML 文档

加载外部资源

HTML 文档通常包含对 外部资源（例如图像、样式表和脚本）的引用。浏览器会从目标服务器下载这些资源，并将其整合到网页中。

渲染网页

当所有必需的资源都加载完成后，浏览器会 渲染 网页。渲染过程涉及将 HTML 文档、CSS 样式表和 JavaScript 代码转换为视觉元素，并将其显示在浏览器窗口中。

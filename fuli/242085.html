火狐浏览器点击书签打开新网页

步骤 1：创建书签

1. 打开您要创建书签的网页。

3. 在弹出的菜单中，为书签命名并选择一个文件夹。

4. 单击“保存”以创建书签。

步骤 2：打开新网页

1. 单击窗口左上角的“书签”图标。

2. 找到您要打开的书签。

3. 单击书签以在新标签页中打开网页。

打开新窗口或新标签页

您可以选择在新的浏览器窗口或标签页中打开网页：

在新窗口中打开：右键单击书签并选择“在新窗口中打开链接”。

在新标签页中打开：右键单击书签并选择“在新标签页中打开链接”。

快捷键

以下快捷键也可以用于打开书签：

打开新标签页中打开书签：Ctrl + Shift + B (Windows) / Cmd + Shift + B (Mac)

打开新窗口中打开书签：Ctrl + Shift + N (Windows) / Cmd + Shift + N (Mac)

额外提示

您可以在书签菜单中重命名、编辑或删除书签。

您可以使用搜索框搜索书签。

您可以将书签整理到文件夹中以保持井井有条。

您还可以将书签从火狐浏览器导出到其他浏览器或设备。

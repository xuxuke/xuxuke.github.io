添加书签

1. 打开您要创建书签的网站。

3. 单击星形图标以打开书签菜单。

4. 在 "名称" 字段中输入书签的名称。

5. 选择要将书签保存到的文件夹。

6. 单击 "保存"。

编辑书签

1. 在书签栏中找到您要编辑的书签。

2. 右键单击书签。

3. 选择 "编辑"。

4. 在 "名称" 字段中编辑书签的名称。

5. 选择要将书签保存到的文件夹。

6. 单击 "保存"。

删除书签

1. 在书签栏中找到您要删除的书签。

2. 右键单击书签。

3. 选择 "删除"。

4. 单击 "确定" 以确认删除。

创建书签文件夹

1. 在书签栏中右键单击。

2. 将鼠标悬停在 "添加" 上。

3. 选择 "文件夹"。

4. 在 "名称" 字段中输入文件夹的名称。

5. 单击 "确定"。

将书签移动到文件夹

1. 在书签栏中找到您要移动的书签。

2. 将书签拖放到书签文件夹中。

导入和导出书签

导入书签

1. 打开您的浏览器。

2. 导航到书签管理器。

3. 单击 "导入书签" 按钮。

4. 选择您要导入的书签文件。

5. 单击 "打开"。

导出书签

1. 打开您的浏览器。

2. 导航到书签管理器。

3. 单击 "导出书签" 按钮。

4. 选择您要导出的书签文件类型。

5. 单击 "保存"。

导航密码大全

基本导航密码

前往书签管理器： Ctrl + B

打开新标签页： Ctrl + T

关闭当前标签页： Ctrl + W

后退： Alt + ←

前进： Alt + →

向上滚动： Space

向下滚动： Shift + Space

查看浏览历史记录： Ctrl + H

清除浏览数据： Ctrl + Shift + Del

选项卡管理

打开新选项卡： Ctrl + T / Ctrl + Shift + T（重新打开关闭的选项卡）

关闭当前选项卡： Ctrl + W

切换到下一个选项卡： Ctrl + Tab

切换到上一个选项卡： Ctrl + Shift + Tab

固定选项卡： 右键单击选项卡并选择“固定”

从固定项中解固定选项卡： 右键单击固定项中的选项卡并选择“从固定项中解固定”

页面控制

刷新页面： F5

停止加载： Esc

放大页面： Ctrl + +

缩小页面： Ctrl + -

重置页面缩放： Ctrl + 0

开发工具

打开开发者工具： F12

查看源代码： Ctrl + U

调试 JavaScript： F8（设置断点）/ F9（继续）

检查元素： 右键单击元素并选择“检查”

其他有用密码

保存页面为 PDF： Ctrl + P

将页面添加到书签： Ctrl + D

复制链接： Ctrl + C

粘贴链接： Ctrl + V

查找页面中的文本： Ctrl + F

进入隐藏界面设置

进入方法

Windows

1. 打开注册表编辑器（regedit）。

2. 导航到以下项：

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

3. 创建一个名为 ShowHiddenFilesAndFolders 的新 DWORD 值。

4. 将其值设置为 1。

Mac

1. 在 Finder 中，按 Command + Shift + G 打开转到文件夹窗口。

2. 输入以下路径：

~/Library/Preferences/com.apple.finder

3. 找到名为 AppleShowAllFiles 的文件。

4. 将其内容复制到文本编辑器中。

5. 找到以下行并在其前面添加一个空格：

TRUE

6. 将更改后的文件保存回其原始位置。

恢复原有设置

Windows

1. 打开注册表编辑器（regedit）。

2. 导航到以下项：

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

3. 删除 ShowHiddenFilesAndFolders 值。

Mac

1. 在 Finder 中，按 Command + Shift + G 打开转到文件夹窗口。

2. 输入以下路径：

~/Library/Preferences/com.apple.finder

3. 找到名为 AppleShowAllFiles 的文件。

4. 将其内容复制到文本编辑器中。

5. 找到以下行并删除其前面的空格：

TRUE

6. 将更改后的文件保存回其原始位置。
